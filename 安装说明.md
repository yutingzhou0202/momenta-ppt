# Momenta PPT Skill 安装说明

> 安装完成后，直接用自然语言告诉 Claude 做 PPT 即可，无需学习任何命令。

---

## 前提条件

- 已安装 **Claude Code**（命令行工具）
- 已安装 **Python 3**（Mac 系统一般自带；Windows 可从 [python.org](https://www.python.org/downloads/) 下载）
- 电脑上有 **PowerPoint**（用于打开生成的文件）

---

## 第一步：安装 Skill 文件

将 `SKILL.md` 放到 Claude Code 的 skills 目录：

**macOS / Linux**
```bash
mkdir -p ~/.claude/skills/momenta-ppt
# 将 SKILL.md 复制到该目录
cp SKILL.md ~/.claude/skills/momenta-ppt/SKILL.md
```

**Windows**
```
%USERPROFILE%\.claude\skills\momenta-ppt\
```
在该目录下放入 `SKILL.md` 文件。

---

## 第二步：下载 PPT 模板

打开以下链接，下载模板文件：

**https://momenta.feishu.cn/wiki/M9rjwsMjCimCapkXvcYcnV5VnhZ**

下载后，将文件放到以下目录（**目录不存在请手动创建**）：

| 系统 | 目标路径 |
|------|---------|
| macOS | `~/Desktop/Claude/` |
| Windows | `C:\Users\<你的用户名>\Desktop\Claude\` |

确保目录中包含：
- `PPT模板.pptx`
- `Momenta PPT模板英文EN.pptx`

---

## 完成！开始使用

打开 Claude Code，直接说：

```
帮我做一个关于 XX 的 PPT，共 5 页
```

首次运行会自动安装所需 Python 依赖（`python-pptx`），约需 10 秒，之后无需重复安装。

---

## 常见问题

**Q：运行脚本报"找不到模板文件"？**
按提示将模板文件放到指定路径后重新运行即可。

**Q：Mac 上中文显示为苹方字体而非微软雅黑？**
正常现象。Skill 在 Mac 上自动使用苹方（PingFang SC）替代，视觉效果等同。

**Q：提示 `python-pptx` 未安装？**
脚本会自动安装，若自动安装失败，手动运行：
```bash
pip3 install python-pptx
```

**Q：生成的 .pptx 无法打开，提示 PermissionError？**
关闭 PowerPoint 中已打开的同名文件后重新运行。
