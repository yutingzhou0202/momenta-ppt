# Momenta PPT Skill 安装说明

> 安装完成后，直接用自然语言告诉 Claude 做 PPT 即可，无需学习任何命令。

---

## 前提条件

- 已安装 **Claude Code**（命令行工具）
- 已安装 **Python 3**（Mac 系统一般自带；Windows 可从 [python.org](https://www.python.org/downloads/) 下载）
- 电脑上有 **PowerPoint**（用于打开生成的文件）

---

## 安装（仅需一步）

将 Skill 仓库克隆到 Claude Code 的 skills 目录：

**macOS / Linux**
```bash
git clone https://devops.momenta.works/Momenta/public/_git/solution_skills \
  ~/.claude/skills/momenta-ppt
```

**Windows（在 Git Bash 中运行）**
```bash
git clone https://devops.momenta.works/Momenta/public/_git/solution_skills \
  "$USERPROFILE/.claude/skills/momenta-ppt"
```

---

## 完成！开始使用

打开 Claude Code，直接说：

```
帮我做一个关于 XX 的 PPT，共 5 页
```

**首次运行时会自动完成：**
- 安装所需 Python 依赖（`python-pptx`）
- 将 PPT 模板复制到工作目录

全程无需手动操作。

---

## 更新 Skill

Skill 每次触发时会自动拉取最新版本，无需手动更新。

如需手动更新：
```bash
cd ~/.claude/skills/momenta-ppt && git pull
```

---

## 常见问题

**Q：提示缺少模板文件？**
确认已按上方步骤完成 `git clone`，模板文件已包含在仓库中。

**Q：Mac 上中文显示为苹方字体而非微软雅黑？**
正常现象。Skill 在 Mac 上自动使用苹方（PingFang SC）替代，视觉效果等同。

**Q：生成的 .pptx 无法打开，提示 PermissionError？**
关闭 PowerPoint 中已打开的同名文件后重新运行。
